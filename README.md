# VideoMediaPlayer

**项目描述：一款简单的视频播放工具,目前支持的视频格式有： mp4，avi，mov 等常见格式。**

## 项目信息
**开发工具：Qt Creator**

**编译环境：Qt_5_12_3_MinGW_64_bit-Debug**

**运行环境：windows**

**依赖项：需提前在windows中配置LAVFilters视频解码器，否则可能无法播放视频,你需要提前完成以下步骤：**

    1)：以管理员身份运行cmd命令行，并进入项目目录LAVFilters下。

    2)：运行以下命令：install_splitter.bat、install_video.bat 和 install_audio.bat
